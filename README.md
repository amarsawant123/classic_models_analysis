# classic_models_analysis

This project presents an interactive Power BI Sales Dashboard developed to analyze sales performance, profitability, customer trends, and product insights across different years and product categories.

The dashboard provides a comprehensive business overview using KPIs, charts, filters, and visual analytics to help stakeholders make data-driven decisions.

The dashboard includes:

Total Sales Analysis
Profit Margin Tracking
Top & Bottom Selling Products
Sales by Product Line
Quantity Ordered Analysis
Year-wise Performance Comparison
Interactive Filters and Slicers
